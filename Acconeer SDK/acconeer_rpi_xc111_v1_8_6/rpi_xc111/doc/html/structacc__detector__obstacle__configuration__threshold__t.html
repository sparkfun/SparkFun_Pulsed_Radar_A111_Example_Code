Detailed Description

Threshold parameters for non coherent background cancellation.

Parameters
stationaryThe amplitude limit for stationary objects close to the sensor
movingThe amplitude limit for moving objects and objects far from the sensor
distance_limit_farFor distances larger than the far limit, use the moving threshold
close_additionThe amplitude increase at closest distance, applied for all velocities
distance_limit_nearclose_addition threshold is applied from first distance in the range and linearly falls off to 0 at first distance plus distance_limit_near
