◆ acc_hal_sensor_wait_for_interrupt_function_t

typedef bool(* acc_hal_sensor_wait_for_interrupt_function_t) (acc_sensor_id_t sensor_id, uint32_t timeout_ms)

Definition of a wait for sensor interrupt function.

This function shall wait at most timeout_ms for the interrupt to become active and then return true. It may return true immediately if an interrupt have occurred since last call to this function.

If waited more than timeout_ms for the interrupt to become active it shall return false.

Note that this function can be called with a timeout_ms = 0.
